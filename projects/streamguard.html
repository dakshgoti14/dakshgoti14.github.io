StreamGuard — Real-Time Fraud Detection Pipeline

  • Tech Stack: Python, Apache Kafka, Spark Streaming, scikit-learn, Redis, FastAPI, AWS, Docker, Kubernetes
  • Github URL: Project Link

Designed a real-time streaming pipeline that scores 1M+ transaction events per hour for fraud in milliseconds, powered by fault-tolerant Kafka topics.

Trained and deployed an anomaly-detection model that flags suspicious activity at ~98% precision, with Redis-backed feature lookups for sub-second scoring.

Built an alerting service that automatically flags and blocks high-risk transactions, exposing live metrics through a FastAPI monitoring dashboard.

Containerized the services with Docker & Kubernetes and deployed on AWS for horizontal scalability and high availability under bursty traffic.
